# 🎆 New Year Wish Generator

A web application to generate personalized New Year wishes with live fireworks and instant sharing.

## 🚀 Features
- Personalized wishes (name, relation, profession, tone)
- Live fireworks animation
- Shareable links & WhatsApp integration
- Secure HTTPS via Cloudflare Tunnel
- Clean, premium UI

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Gunicorn & Nginx
- Cloudflare Tunnel
- Linux (Ubuntu)

## 📸 Live Demo
🔗 https://shipment-horses-sri-given.trycloudflare.com/

## 📌 Purpose
Built as a hands-on project to practice:
- Real-world deployment
- Reverse proxy & HTTPS
- Production debugging
- DevOps fundamentals
